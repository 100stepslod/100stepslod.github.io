DATA
CONNECTION

We selected eight entities of different types to create our RDF triples. The selection criteria were the deep interconnection among these entities, which allowed us to use them both as subjects and objects of the triples. We defined specific URIs for the core concepts of the project domain, whilst we mainly used existing vocabularies, authority controls, and ontologies for their description. Examples include FOAF, RIC‑O, GeoNames, VIAF and LCSH.

  • The table representing the triples related to the selected entities is downloadable in CSV format from the link below.
  • In the second section, we used the triples to create an RDF model using the Turtle syntax.
  • Finally, we provided a visual representation of the connection among the triples describing the data.
